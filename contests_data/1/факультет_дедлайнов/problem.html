Климентий учится на Факультете Дедлайнов. Семестр длится $$$m$$$ дней, в течение семестра Климентию предстоит выполнить $$$n$$$ заданий.

Задание с номером $$$i$$$ нужно выполнить с дня $$$l_i$$$ до дня $$$r_i$$$ включительно. Задание можно выполнить в любой день $$$p$$$ $$$\left(l_i \leq p \leq r_i\right)$$$.

Также для каждого дня $$$p = 1, 2, \ldots, m$$$ известна продуктивность Климентия, которая выражается целым неотрицательным числом $$$a_p$$$.

Массив значений продуктивности $$$a_1, a_2, \ldots, a_m$$$ задан следующим рекуррентным соотношением:

$$$\bullet$$$ $$$a_1 = x$$$

$$$\bullet$$$ Для всех $$$k = 2, 3, \ldots, m$$$ $$$a_k = \left(a_{k - 1} \cdot b + c\right) \% mod$$$, где $$$\%$$$ — операция взятия остатка от деления, а $$$mod$$$ — число, остаток от деления на которое вычисляется.

Климентий хочет посвятить выполнению заданий только один день в семестре. Необходимо выбрать такой день $$$p$$$, чтобы произведение $$$cnt(p) \cdot a_p$$$ было максимально. Здесь $$$cnt(p)$$$ — количество заданий, которые можно сдать в день $$$p$$$.

Определите максимальное произведение $$$cnt(p) \cdot a_p$$$.

Входные данные

В первой строке вводятся два целых числа $$$n, m$$$ — количество заданий и количество дней в семестре $$$\left(1 \leq n \leq 2 \cdot 10^5, 1 \leq m \leq 2 \cdot 10^9\right)$$$.

Во второй строке вводятся четыре целых числа $$$x, b, c, mod$$$, определяющие массив продуктивностей $$$a$$$ $$$\left(0 \leq x \lt mod, 0 \leq b, c \leq 10^9, 1 \leq mod \leq 2 \cdot 10^5\right)$$$.

В следующих $$$n$$$ строках вводятся границы дней для выполнения каждого из $$$n$$$ заданий. Каждая граница описывается двумя числами $$$l_i, r_i$$$ $$$\left(1 \leq l_i \leq r_i \leq m\right)$$$.

Выходные данные

В единственной строке выведите одно целое число — максимальное значение $$$cnt(p) \cdot a_p$$$.

Система оценки

Решения, работающие для ограничений $$$n, m \le 2000$$$, наберут не меньше $$$20$$$ баллов

Решения, работающие для ограничений $$$m \le 2 \cdot 10^5$$$, наберут не меньше $$$40$$$ баллов

Решения, работающие для ограничений $$$n, mod \le 2000$$$, наберут не меньше $$$60$$$ баллов

Решения, работающие для ограничений $$$mod \le 2000$$$, наберут не меньше $$$74$$$ баллов

Решения, работающие без дополнительных ограничений, наберут $$$100$$$ баллов

Примечание

$$$\bullet$$$ Рассмотрим первый пример

Массив продуктивностей выглядит следующим образом:

$$$1, 5, 6, 1, 5, 6, 1, 5, 6, 1$$$

Для каждого $$$p = 1, 2, \ldots, 10$$$ количество заданий, которые можно сдать в день $$$p$$$, равно:

$$$1, 2, 2, 1, 2, 2, 1, 1, 1, 1$$$

И для каждого $$$p = 1, 2, \ldots, 10$$$ произведение $$$cnt(p) \cdot a_p$$$ равно:

$$$1, 10, 12, 1, 10, 12, 1, 5, 6, 1$$$

Таким образом, максимальный ответ равен $$$12$$$.

$$$\bullet$$$ Рассмотрим второй пример

Массив продуктивностей выглядит следующим образом:

$$$3, 5, 4, 0, 2, 1, 6, 8, 7, 3, 5, 4$$$

Для каждого $$$p = 1, 2, \ldots, 12$$$ количество заданий, которые можно сдать в день $$$p$$$, равно:

$$$2, 3, 3, 4, 3, 4, 4, 4, 4, 4, 3, 3$$$

И для каждого $$$p = 1, 2, \ldots, 12$$$ произведение $$$cnt(p) \cdot a_p$$$ равно:

$$$6, 15, 12, 0, 6, 4, 24, 32, 28, 12, 15, 12$$$

Таким образом, максимальный ответ равен $$$32$$$.
