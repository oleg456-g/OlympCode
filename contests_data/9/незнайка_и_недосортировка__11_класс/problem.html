Знайка ввёл в Цветочном городе буквенную систему счисления — позиционную систему с основанием 52, в которой цифрами служат заглавные и строчные латинские буквы. Таблица цифр:


Пример: 202510 = 38·521 + 49·520 = mx.

Все стали пользоваться буквенной системой счисления, кроме Незнайки. Он разрешил переполнять разряды — записывать в них цифры со значениями, превышающими 51, оставив основание 52. Незнайка дополнил таблицу цифр с помощью модификаторов:

  • ^ (крышка) — добавляет 52 к значению буквы: A^ = 52, B^ = 53, …, z^ = 103
  • ~ (волна) — добавляет 104: A~ = 104, B~ = 105, …, z~ = 155
  • _ (подчёркивание) — добавляет 156: A_ = 156, B_ = 157, …, z_ = 207

Таким образом, одно и то же число можно записать по-разному. Например:


(здесь mx: 38·52+49, lx^: 37·52+101, kx~: 36·52+153, jx_: 35·52+205)

Знайка предложил Незнайке отсортировать по убыванию последовательность различных чисел в модифицированной системе. Незнайка допустил ровно одну ошибку: в последовательности A[i] нашлась одна пара элементов A[i1] и A[i2] (i1 ≠ i2), стоящих не на своих местах. После обмена местами A[i1] и A[i2] последовательность становится убывающей.

Найдите пару индексов для обмена. Во вводе буква с крышкой задаётся двумя символами: A^ — «A с крышкой» (значение 52). Буква с волной: B~ — «B с волной» (значение 105). Буква с подчёркиванием: C_ — «C с подчёркиванием» (значение 158).

Формат ввода

В первой строке вводится натуральное число N: 1 < N < 501. В следующих N строках содержатся записи различных неотрицательных чисел A[i] (i = 1..N) в модифицированной системе счисления. В каждой записи используются только заглавные и/или строчные латинские буквы, крышка ^, волна ~, подчёркивание _. Длина каждой записи не более 50 символов.

В единственной строке выводятся искомые номера i1 и i2 (i1 < i2), разделённые одиночным пробелом. Нумерация с единицы.
