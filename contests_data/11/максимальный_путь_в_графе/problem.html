В древнем городе между башнями существует сеть тайных подземных туннелей. По этим туннелям перемещаются курьеры.

Каждый туннель соединяет две башни и имеет время прохождения.

Иногда курьеры передают особо важные сообщения. В таком случае они хотят выбрать маршрут из башни 1 в башню N так, чтобы время пути было минимальным.

Однако существует дополнительное правило безопасности.

Курьер может пройти не более K туннелей подряд. После этого он обязан сделать остановку на одной из башен, чтобы проверить, не следят ли за ним.

Остановка занимает S секунд.

После остановки курьер может снова пройти до K туннелей подряд.

Формат ввода

На первой строке записаны четыре числа N M K S:

  • N — количество башен (2 ≤ N ≤ 100000),
  • M — количество туннелей (1 ≤ M ≤ 200000),
  • K — максимальное количество туннелей подряд без остановки (1 ≤ K ≤ 50),
  • S — время одной остановки (0 ≤ S ≤ 109).

Далее записаны M строк, каждая содержит три числа u v w — между башнями u и v есть туннель со временем прохождения w (1 ≤ u, v ≤ N, 1 ≤ w ≤ 109).

Туннели двусторонние. Гарантируется, что существует хотя бы один путь между башнями 1 и N.

Формат вывода

Выведите одно число — минимальное время, за которое курьер может добраться из башни 1 в башню N.
