Дано число $$$N$$$. Назовём неравенством набора $$$a_{1},...,a_{n}$$$ сумму модулей попарных разностей (то есть значение $$$ \sum_{i=1}^{n}\sum_{j=i+1}^{n}|a_{i}-a_{j}| $$$).

Найдите минимальное неравенство набора, удовлетворяющего следующим условиям:

  • Произведение чисел в наборе равняется $$$N$$$
  • В наборе есть хотя бы два элемента
  • Все элементы в наборе — натуральные числа, больше либо равные двум

Если такого набора не существует, выведите -1.

Входные данные

В первой строке задано количество независимых запросов $$$t$$$. В следующих $$$t$$$ строках даны запросы, каждый из которых содержит одно целое число $$$N$$$.

В первом тесте $$$t=3000$$$, при этом $$$N \le 10^{10}$$$. Во втором тесте $$$t=7000$$$, при этом $$$N \le 10^{16}$$$

Выходные данные

Для каждого запроса выведите одно число в отдельной строке — ответ при данном $$$N$$$.
