# Выбор заявки

| Ограничение по времени | Ограничение по памяти | Ввод | Вывод |
| :--- | :--- | :--- | :--- |
| 1.0 секунд | 256 МБ | стандартный ввод (stdin) | стандартный вывод (stdout) |

## Условие задачи
Дана лекционная аудитория, в которой несколько профессоров хотят прочесть свои лекции. Для составления расписания профессора подали заявки вида $[s_i; f_i)$ — время начала и конца лекции. Лекция считается открытым полуинтервалом, то есть какая-то лекция может начаться в момент окончания другой без перерыва. 

Составьте расписание занятий так, чтобы выполнить максимальное количество заявок.

## Формат входных данных
В первой строке вводится натуральное число $n$ ($1 \le n \le 1000$) — общее количество заявок. Затем вводится $n$ строк с описаниями заявок — по два числа в каждой: $s_i$ и $f_i$.

Гарантируется, что $s_i < f_i$. Время начала и окончания лекции — натуральные числа, не превышающие 1440 (в минутах с начала суток).

## Формат выходных данных
Выведите одно число — максимальное количество заявок, которые можно выполнить.

## Примеры

| Стандартный ввод | Стандартный вывод |
| :--- | :--- |
| `1`<br>`5 10` | `1` |
| `3`<br>`1 5`<br>`2 3`<br>`3 4` | `2` |

## Примечания
Во втором примере можно выполнить вторую и третью заявки.
