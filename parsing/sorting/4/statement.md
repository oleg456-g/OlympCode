# Результаты олимпиады

| Ограничение по времени | Ограничение по памяти | Ввод | Вывод |
| :--- | :--- | :--- | :--- |
| 1.0 секунд | 256 МБ | стандартный ввод (stdin) | стандартный вывод (stdout) |

## Условие задачи
Во время проведения олимпиады каждый из участников получил свой идентификационный номер — натуральное число. Необходимо отсортировать список участников олимпиады по количеству набранных ими баллов.

## Формат входных данных
На первой строке дано число $n$ ($1 \le n \le 100\,000$) — количество участников.  
На каждой из следующих $n$ строк даны два целых числа: идентификационный номер и набранное количество баллов соответствующего участника. Все числа во входных данных положительные и не превышают $10^5$.

## Формат выходных данных
Выведите исходный список участников в порядке убывания баллов (каждая пара «идентификатор балл» в отдельной строке).  
Если у некоторых участников количество баллов совпадает, то между собой их нужно упорядочить в порядке возрастания идентификационного номера.

## Примеры

| Стандартный ввод | Стандартный вывод |
| :--- | :--- |
| `3`<br>`101 80`<br>`305 90`<br>`200 14` | `305 90`<br>`101 80`<br>`200 14` |
| `3`<br>`20 80`<br>`30 90`<br>`25 90` | `25 90`<br>`30 90`<br>`20 80` |
