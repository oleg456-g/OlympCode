# Сортировка подсчетом

| Ограничение по времени | Ограничение по памяти | Ввод | Вывод |
| :--- | :--- | :--- | :--- |
| 1.0 секунд | 256 МБ | стандартный ввод (stdin) | стандартный вывод (stdout) |

## Условие задачи
Реализуйте алгоритм сортировки подсчетом для целых чисел.

## Формат входных данных
В первой строке задано число $n$ ($1 \le n \le 10^5$) — количество элементов в массиве.  
В следующей строке входных данных расположены сами элементы массива — целые числа $A_i$, по модулю не превосходящие $10^4$ (т.е. $-10\,000 \le A_i \le 10\,000$).

## Формат выходных данных
Выведите отсортированный по неубыванию массив. Элементы выводите через пробел.

## Примеры

| Стандартный ввод | Стандартный вывод |
| :--- | :--- |
| `5`<br>`1 3 4 2 5` | `1 2 3 4 5` |
| `4`<br>`10 -5 -5 0` | `-5 -5 0 10` |
