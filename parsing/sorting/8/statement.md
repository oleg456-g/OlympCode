# Сортировка слиянием

| Ограничение по времени | Ограничение по памяти | Ввод | Вывод |
| :--- | :--- | :--- | :--- |
| 1.0 секунд | 256 МБ | стандартный ввод (stdin) | стандартный вывод (stdout) |

## Условие задачи
Отсортируйте заданный массив целых чисел по неубыванию, используя алгоритм **«сортировки слиянием»**.

## Формат входных данных
В первой строке входных данных задано одиночное число $n$ ($1 \le n \le 10^5$) — размер исходного массива.  
Во второй строке задано $n$ целых чисел $A_i$, разделенных пробелами. Каждое число по модулю не превосходит $10^9$ (т.е. $-10^9 \le A_i \le 10^9$).

## Формат выходных данных
Выведите элементы отсортированного массива в одну строку через пробел.

## Примеры

| Стандартный ввод | Стандартный вывод |
| :--- | :--- |
| `2`<br>`3 1` | `1 3` |
| `5`<br>`4 2 1 3 5` | `1 2 3 4 5` |
