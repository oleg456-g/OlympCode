# Анаграммы

| Ограничение по времени | Ограничение по памяти | Ввод | Вывод |
| :--- | :--- | :--- | :--- |
| 1.0 секунд | 256 МБ | стандартный ввод (stdin) | стандартный вывод (stdout) |

## Условие задачи
Слово называется анаграммой другого слова, если оно может быть получено перестановкой его символов. Напишите программу, которая проверяет, являются ли два заданных слова анаграммами друг друга.

## Формат входных данных
Входные данные содержат две строки. На первой строке записано первое слово, на второй строке — второе слово. Слова состоят только из строчных латинских букв и цифр. Длина каждого слова не превышает $10^5$ символов.

## Формат выходных данных
Выведите строку `YES`, если введённые слова являются анаграммами друг друга, и `NO` — в противном случае.

## Примеры

| Стандартный ввод | Стандартный вывод |
| :--- | :--- |
| `sharm`<br>`marsh` | `YES` |
| `ananas`<br>`nnaass` | `NO` |
