# Разброс

| :--- | :--- | :--- | :--- |

## Условие задачи
Дан массив из $n$ целых чисел. Так получилось, что в наборе не будет двух элементов, разность между которыми превышает $10^7$. Отсортируйте массив по неубыванию.

## Формат входных данных
Первая строка содержит целое число $n$ ($1 \le n \le 10^5$) — количество элементов в массиве.  
Во второй строке задано $n$ целых чисел $A_i$, каждое из которых по модулю не превосходит $2 \cdot 10^9$. Гарантируется, что для любых двух элементов массива $A_i$ и $A_j$ выполняется условие: $|A_i - A_j| \le 10^7$.

## Формат выходных данных
Выведите $n$ заданных чисел в порядке неубывания через пробел.
