# Перевозки на беспилотниках

| Ограничение по времени | Ограничение по памяти | Ввод | Вывод |
| :--- | :--- | :--- | :--- |
| 1.0 секунд | 256 МБ | стандартный ввод (stdin) | стандартный вывод (stdout) |

## Условие задачи
Известная всем компания на букву Я решила расширять сферу своей деятельности и поэтому открыла отдел, занимающийся перевозкой грузов по городу с помощью машин-беспилотников. Так как компания Я использует только последние разработки во всех своих проектах, то решено создать сервис, который распределяет машины-беспилотники по заказам.

Одна машина может совершить одну доставку за день. Доставка осуществляется по городу, который представляет собой сетку из $n \times m$ домов. Карта задана матрицей размера $n \times m$. В дом на пересечении $i$-й строки и $j$-го столбца надо отвезти определенный груз, стоимость доставки которого равна $a_{ij}$.

В каждую машину запрограммирована «жадность», поэтому она будет совершать доставку только в том случае, если стоимость доставки не меньше уровня ее жадности.

Ваша задача — определить максимальное количество заказов, которые получится выполнить.

## Формат входных данных
В первой строке входных данных заданы числа $n$ и $m$ ($1 \le n, m \le 40$) — размеры карты города.  
В каждой из последующих $n$ строк содержится по $m$ целых чисел, обозначающих стоимость доставки груза в соответствующий дом.

В $(n + 2)$-й строке находится число $k$ ($1 \le k \le 2000$) — количество доступных машин-беспилотников.  
В $(n + 3)$-й строке содержатся $k$ целых чисел, $i$-е из которых обозначает уровень жадности $i$-го беспилотника.

Все стоимости доставок и уровни жадности — натуральные числа, не превышающие $10^9$.

## Формат выходных данных
Выведите единственное число — максимальное количество заказов, которое удастся выполнить.
