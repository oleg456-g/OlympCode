# Сортировка выбором

| Ограничение по времени | Ограничение по памяти | Ввод | Вывод |
| :--- | :--- | :--- | :--- |
| 1.0 секунд | 256 МБ | стандартный ввод (stdin) | стандартный вывод (stdout) |

## Условие задачи
Отсортируйте массив целых чисел по неубыванию (каждый последующий элемент не меньше предыдущего), используя алгоритм **«выбора максимума»**.

## Формат входных данных
В первой строке входных данных задано одиночное число $n$ ($1 \le n \le 1000$) — размер исходного массива.  
Во второй строке задано $n$ целых чисел $A_i$, разделенных пробелами. Каждое число по модулю не превосходит $1000$ (т.е. $-1000 \le A_i \le 1000$).

## Формат выходных данных
Выведите элементы отсортированного массива в одну строку через пробел.

## Примеры

| Стандартный ввод | Стандартный вывод |
| :--- | :--- |
| `5`<br>`5 4 3 2 1` | `1 2 3 4 5` |
| `3`<br>`10 -5 10` | `-5 10 10` |
