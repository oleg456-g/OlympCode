# Оценка

| Ограничение по времени | Ограничение по памяти | Ввод | Вывод |
| :--- | :--- | :--- | :--- |
| 1.0 секунд | 256 МБ | стандартный ввод (stdin) | стандартный вывод (stdout) |

## Условие задачи
В школе у Васи выставляют очень (ну очень) длинные оценки. Каждая оценка занимает целую полоску бумаги и является каким-то целым неотрицательным числом. Младшая сестра Васи Настя влетела в комнату и начала учиться обращаться с ножницами. На оценке Васи, конечно же.

В итоге у Васи осталось несколько более мелких полосок с цифрами. Но любую ситуацию надо обращать в свою сторону, поэтому Вася решил склеить эти полоски в таком порядке, чтобы получившийся результат был наибольшим из всех возможных. Потом Вася покажет эту оценку родителям, чтобы ему разрешили писать раунды на Codeforces вместо школы.

Помогите Васе! Выведите максимальную оценку, которую он может составить.

## Формат входных данных
Входные данные состоят из одной или более строк. Каждая строка содержит последовательность цифр, представляющую собой один из кусочков разрезанной оценки. 

Количество строк не превышает $100$, каждая строка содержит от $1$ до $100$ цифр. Гарантируется, что хотя бы в одной строке первая цифра отлична от нуля.

## Формат выходных данных
Выведите одну строку — максимальное число, которое можно склеить из данных кусков.

## Примеры

| Стандартный ввод | Стандартный вывод |
| :--- | :--- |
| `2`<br>`20`<br>`004`<br>`66` | `66220004` |
| `3`<br>`30` | `330` |
