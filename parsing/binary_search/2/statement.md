# Приближенный двоичный поиск

| Ограничение по времени | Ограничение по памяти | Ввод | Вывод |
| :--- | :--- | :--- | :--- |
| 1.0 секунд | 256 МБ | стандартный ввод (stdin) | стандартный вывод (stdout) |

## Условие задачи
Реализуйте алгоритм приближенного бинарного поиска.

## Формат входных данных
В первой строке входных данных содержатся два целых числа $n$ и $k$ ($1 \le n, k \le 10^5$) — размер отсортированного массива и количество запросов соответственно.  
Во второй строке задаются $n$ целых чисел первого массива, отсортированного по неубыванию.  
В третьей строке задаются $k$ целых чисел второго массива (запросы).  

Каждое число в обоих массивах по модулю не превосходит $2 \cdot 10^9$.

## Формат выходных данных
Для каждого из $k$ чисел второго массива выведите в отдельной строке число из первого массива, наиболее близкое к данному. Если таких чисел несколько, выведите меньшее из них.

## Примеры

| Стандартный ввод | Стандартный вывод |
| :--- | :--- |
| `5 5`<br>`1 3 5 7 9`<br>`2 4 8 1 6` | `1`<br>`3`<br>`7`<br>`1`<br>`5` |
