# Квадратный корень и квадратный квадрат

| Ограничение по времени | Ограничение по памяти | Ввод | Вывод |
| :--- | :--- | :--- | :--- |
| 1.0 секунд | 256 МБ | стандартный ввод (stdin) | стандартный вывод (stdout) |

## Условие задачи
Найдите такое вещественное число $x$, что $x^2 + \sqrt{x} = C$, с точностью не менее 6 знаков после запятой.

## Формат входных данных
В единственной строке содержится вещественное число $C$ ($1 \le C \le 10^{10}$).

## Формат выходных данных
Выведите одно число — искомый корень $x$. Ваш ответ будет засчитан, если его абсолютная или относительная погрешность не превосходит $10^{-6}$.

## Примеры

| Стандартный ввод | Стандартный вывод |
| :--- | :--- |
| `2.0000000000` | `1.0000000000` |
| `18.0000000000` | `4.0000000000` |
