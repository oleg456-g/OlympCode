# Провода

| Ограничение по времени | Ограничение по памяти | Ввод | Вывод |
| :--- | :--- | :--- | :--- |
| 1.0 секунд | 256 МБ | стандартный ввод (stdin) | стандартный вывод (stdout) |

## Условие задачи
Дано $n$ отрезков провода длиной $l_1, l_2, \dots, l_n$ сантиметров. Требуется с помощью разрезания получить из них $k$ равных отрезков как можно большей длины, выражающейся целым числом сантиметров. Из одного исходного отрезка можно получить несколько меньших отрезков, но нельзя склеивать два или более отрезка в один. Остатки проводов утилизируются.

Если нельзя получить $k$ отрезков длиной даже 1 см, выведите 0.

## Формат входных данных
В первой строке входных данных заданы целые числа $n$ ($1 \le n \le 10^4$) и $k$ ($1 \le k \le 10^4$).  
В следующих $n$ строках заданы длины исходных проводов $l_i$ ($100 \le l_i \le 10^7$) — по одному числу в строке.

## Формат выходных данных
Выведите одно целое число — максимальную возможную длину полученных отрезков.
