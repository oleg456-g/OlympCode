# Массивы

| Ограничение по времени | Ограничение по памяти | Ввод | Вывод |
| :--- | :--- | :--- | :--- |
| 1.0 секунд | 256 МБ | стандартный ввод (stdin) | стандартный вывод (stdout) |

## Условие задачи
Даны два массива целых чисел. Для каждого элемента второго массива определите, сколько раз он встречается в первом массиве.

## Формат входных данных
В первой строке входных данных содержится число $n$ ($1 \le n \le 10^5$) — количество элементов в первом массиве.  
Во второй строке задаются $n$ целых чисел — элементы первого массива.  

В третьей строке содержится число $m$ ($1 \le m \le 10^5$) — количество элементов во втором массиве.  
В четвёртой строке задаются $m$ целых чисел — элементы второго массива.  

Все числа в обоих массивах по модулю не превосходят $10^9$.

## Формат выходных данных
Выведите $m$ чисел через пробел: для каждого элемента второго массива укажите количество его вхождений в первый массив.

## Примеры

| Стандартный ввод | Стандартный вывод |
| :--- | :--- |
| `3`<br>`1 2 1`<br>`4`<br>`0 1 2 3` | `0 2 1 0` |
