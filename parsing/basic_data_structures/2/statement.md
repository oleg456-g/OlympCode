# Постфиксная запись

| Ограничение по времени | Ограничение по памяти | Ввод | Вывод |
| :--- | :--- | :--- | :--- |
| 1.0 секунд | 256 МБ | стандартный ввод (stdin) | стандартный вывод (stdout) |

## Условие задачи
В постфиксной записи (или обратной польской записи) операция записывается после двух операндов. Например, сумма двух чисел $A$ и $B$ записывается как `A B +`. Запись `B C + D *` обозначает привычное нам $(B + C) * D$, а запись `A B C + D * +` означает $A + (B + C) * D$.

Достоинство постфиксной записи в том, что она не требует скобок и дополнительных соглашений о приоритете операторов для своего чтения.

## Формат входных данных
В единственной строке записано выражение в постфиксной записи, содержащее цифры и операции `+`, `-`, `*`. Числа и операции разделяются пробелами. В конце строки может быть произвольное количество пробелов. Числа не превосходят 100 по модулю.

## Формат выходных данных
Необходимо вывести значение записанного выражения.

## Примеры

| Стандартный ввод | Стандартный вывод |
| :--- | :--- |
| `8 9 + 1 7 - *` | `-102` |
